A ROPIS(Retail Online, Pickup In-Store) eCommerce website for a Korean grocery store. This project features a authentication and map integration via KakaoTalk, allowing users to easily log in and find nearby store locations. For image hosting, I used [Cloudinary](https://cloudinary.com/) and for hosting I used [Render](https://render.com/)
